# Python-Practice-with-Pycharm

The Python files in this repository include the codes I practised along with necessary comments and deep explanations. 

I didn't just focus on learning syntax, but asked why and how everytime and learnt a lot. Chatgpt was used for problems, explanations and deep understanding and not for writing code. 

Topics I covered:

1. I understood the difference between is and == really well. Logical operators and boolean values.
 
2. Mutability and Immutability depth.
  
3. And, Or, Not priority and output.
   
4. Truthy and Falsy values in Python using CPython algorithm.

5. Conditional statements like if, else, elif. Different scenarios for using elif and if.

6. Loops like "for" and "while" and nested loops.

7. Functions, Parameters, Arguments, Global and Local variables.

8. Generators.

9. Decorators.

10. OOPS - Inheritance, Polymorphism, Data Abstraction, Encapsulation, Constructor (__init__).

11. I focused on Python Internal Memory Model and hence visualizing OOPS and functions became easier.
